# **App Name**: AI Nyaya Mitra

## Core Features:

- Problem Narration: Allow users to describe their legal problem using voice or text input. Supports multiple languages.
- AI Case Analysis: Use an LLM tool to classify the case, identify relevant laws and sections, and determine the appropriate jurisdiction (police station/court).
- Case Strength Analyzer: Evaluate the strength of the case based on similar past judgments, evidence availability, and other risk factors, providing a strength score.
- Document Generation: Generate legal documents such as police complaints, FIR applications, and legal notices in multiple formats (Simple English, Legal English, Regional Language).
- Police Process Guide: Provide guides on how to file an FIR, what to do if the police refuse an FIR, and explain arrest rules. Uses step-by-step text, voice explanations, and animated videos.
- Lawyer Connect: Connect citizens with lawyers based on location, language preference, and fixed-fee services.
- Case Tracker: Track case hearing dates, document checklists, and court reminders. Provides alerts via WhatsApp and call.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke trust and authority in the legal context.
- Background color: Light Grayish-Blue (#E8EAF6), a very desaturated version of the primary, for a calm and professional feel.
- Accent color: Soft Amber (#FFCA28), an analogous color, used to highlight key actions and information without overwhelming the user.
- Body and headline font: 'PT Sans', a humanist sans-serif, suitable for headlines or body text.
- Note: currently only Google Fonts are supported.
- Use clear, universally understandable icons representing legal concepts.
- Subtle animations and transitions to guide the user through complex legal processes.
